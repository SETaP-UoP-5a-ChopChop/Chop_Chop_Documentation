Introduction
================

Welcome to the documentation for Chop Chop, a react-native and python app that allows users to find recipes, create recipes and store a pantry of ingredients. This documentation will guide you through the installation process, usage, and API reference for the application.

Our front-end uses React Native, which allows us to create a cross-platform mobile application that can run on both iOS and Android devices. The back-end is built using Python, which provides a robust and scalable server to handle API requests and manage data.

Our database uses SQLite, a lightweight and efficient database that is perfect for mobile applications. It allows us to store user data, recipes, and pantry information locally on the device, ensuring fast access and offline functionality.
